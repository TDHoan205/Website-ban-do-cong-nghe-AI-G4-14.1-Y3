Hướng dẫn cài đặt
Ollama (Khuyến nghị - Miễn phí vĩnh viễn)
  1. Tải Ollama từ: ollama.com/download
  2. Sau khi cài xong, chạy: ollama pull llama3.2
  3. Khởi động server: ollama serve
  4. Kiểm tra: curl http://localhost:11434/api/tags
OpenRouter (Cloud - Free $5 credit)
  1. Đăng ký tại: openrouter.ai/credits
  2. Tạo API Key tại: openrouter.ai/keys
  3. Thêm vào file .env: OPENROUTER_API_KEY=sk-or-v1-xxxxx
Groq (Cloud - Rất nhiều Free!)
  1. Đăng ký tại: console.groq.com/keys
  2. Tạo API Key
  3. Thêm vào file .env: GROQ_API_KEY=gsk_xxxxx
Mẹo: Hệ thống sẽ tự động thử theo thứ tự ưu tiên: Ollama → OpenRouter → Groq → Gemini.
Ưu tiên Ollama để không tốn chi phí gì cả!
